# SLI / SLO Model

## SLIs

- Availability
- Latency
- Error rate
- Deployment success rate

## SLOs

- 99.9% API availability
- p95 latency under 300ms
- error rate under 1%
- deployment success above 95%

## Error Budget

- Monthly availability budget for 99.9%: about 43.2 minutes
- Burn alerts should trigger on fast and sustained exhaustion risk
